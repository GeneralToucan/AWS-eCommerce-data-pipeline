# AWS-eCommerce-data-pipeline
AWS Data Engineering Pipeline for eCommerce dataset from Kaggle (AWS Glue, Iceberg, Athena)

## Overview
The project created a data pipeline leveraging AWS Services including but not limited to AWS Glue, Iceberg and Athena.

The dataset is sourced from Kaggle: https://www.kaggle.com/datasets/mkechinov/ecommerce-behavior-data-from-multi-category-store
